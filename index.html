ChartEx Virtual Workbench

This repository is the home of the ChartEx Virtual Workbench, an interactive system for working with medieval charter documents. This tool was created as a demonstrator for the ChartEx project (http://www.chartex.org) by the Human Computer Interaction Research Group in the Department of Computer Science at the University of York.

Technology

The ChartEx Virtual Workbench uses the CodeIgniter platform, using a combination of PHP, HTML5 and Javascript. It also uses a MySQL database for storing the data used by the application.

Setup and use

Setup information is provided in the README.md available in the repository. You will require a web server (e.g. Rocket, Apache) to run the application as well as a MySQL database server. Each of these can be configured to your local repository.
